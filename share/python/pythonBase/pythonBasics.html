  • 什么是 Python ?

    Python 是一种解释型、面向对象、动态数据类型的高级程序设计语言。

    Python 由 Guido van Rossum 于 1989 年底发明，第一个公开发行版发行于 1991 年。像 Perl 语言一样, Python 源代码同样遵循 GPL(GNU General Public License) 协议。

  • Python 的强大之处

    1. 它是人工智能、大数据分析的主要支持语言。
    2. 它被称为“胶水语言”，能够更好的和其他语言及机器硬件粘合在一起。
    3. 它有着庞大的免费的“代码库”，初学者可以随意使用里面资源实现自己功能。
    4. 它是一个脚本语言，和其他语言相比更加简洁、高效。
    5. 它更加通俗易懂，好多省市已经将 Python 加入中小学课程。
    6. 最最重要的是， Python 广泛应用于自动化办公、自动化运维、网站开发、网络爬虫、大数据分析、数据挖掘、科学计算、机器学习、深度学习、神经网络等领域。

  • Python 基础语法1

    print() 函数的用法有以下几种：

    1. 单刀赴会----不带引号。
    2. 黄袍加身----搭配单引号、双引号、三引号

    让我们一睹风采。

    无引号----单刀赴会——不带引号，可直接传入数字、变量名。

      print(1314)
      name = '哪吒'
      print(name)

    单引号用法----黄袍加身。

      print('哪吒')

    这就是 print()中引号的用法: 当括号里有引号的时候，就相当于告诉计算机——您傻傻的就行了，不需要干别的，我给您什么，您就给我打印什么。

    双引号用法----当输出内容中有单引号时，使用双引号可以输出单引号。

      print("哪吒")   #和单引号用法一样
      print("let's go go go!")  #输出为：let's go go go!

    # 号常用作单行注释符号，用于解释单行代码。#号后面的任何内容都不会显示在屏幕上。

    三引号用法 ----- print()函数中使用三引号'''实现跨行输出。

      print('''哪吒，
      托塔李天王家的三太子，
      变化三头六臂又或三头八臂，
      百邪不侵专克摄魂夺魄的莲花化身''')
    输出为(格式保持不变)：
    哪吒，
    托塔李天王家的三太子，
    变化三头六臂又或三头八臂，
    百邪不侵专克摄魂夺魄的莲花化身

    转义字符

    还有另一种方法实现换行，就是使用转义字符 \n,像这样：

      print('哪吒，\n托塔李天王家的三太子，\n变化三头六臂又或三头八臂，\n百邪不侵专克摄魂夺魄的莲花化身'))
    输出为：
    哪吒，
    托塔李天王家的三太子，
    变化三头六臂又或三头八臂，
    百邪不侵专克摄魂夺魄的莲花化身

    转义字符总结：

    转义字符意义
    \a响铃(BEL)
    \b退格(BS)，将当前位置移到前一列
    \f换页(FF)，将当前位置移到下页开头
    \n换行(LF)，将当前位置移到下一行开头
    \r回车(CR)，将当前位置移动到本行开头
    \t水平制表(HT)，跳到下一个TAB位置
    \v垂直制表(VT)
    \0空字符(NUL)
    \'代表一个单引号(撇号)字符
    \"代表一个双引号字符
    \?代表一个问号
    \\代表一个单斜杠字符
    \xhh1到2位十六进制所表示的任意字符
    \ddd1到3位八进制所代表的任意字符

    变量与赋值

      name ='哪吒'

    这就是一个常见的给变量赋值的招式。 在这里 name 就是一个变量，将'哪吒'这两个文字赋值给了 name 这个变量。

    变量的命名规范

    1. 只能包含数字、字母和下划线
    2. 不能以数字开头
    3. 尽量描述包含的数据内容

    除了变量名之外，"="不是等于的意思，而是赋值，表示把 = 号右边的内容放进左边的变量。变量的最终值等于最后一次赋值结果。

  • Python 基础语法2

    数据类型
    在 Python 江湖里，常用的数据类型有三种：字符串，整数，浮点数。

    字符串----穿上单引号、双引号、三引号黄袍的内容就是字符串，无论引号里面内容是中文、英文、法文、数字、符号、甚至是火星文。 就是由一个个字符连接起来的组合，你平时所见的字母、数字、汉字、符号都是字符。
    如：'哪吒'、''美国队长''、'''黑寡妇'''

    整数----整数，英文为 integer，简称 int。Python 江湖的整数和我们现实数学中定义的一样：整数是正整数、零和负整数的统称。说白了，就是没有小数点的数字。
    如：18、192 、80

    浮点数----英文称为 float，Python 江湖中，将带有小数点的纯数字定义为浮点数。它是我们最常见的数据类型之一。
    如：-9.99、9.00、3.1415926

    四则运算----法则和数学上的一致，符号有点差别
    加(+)、减(-)、乘(*)、除(/)
    Python中'%'为取余，例如：num = 3%2 --> num值为1

    字符串的拼接

      hero = '美国队长'
      title = '漫威漫画'
      action = '取材于'
      print(hero+action+title)
    输出：美国队长取材于漫威漫画

    数据类型的查询— type()函数

      name = '美国队长'
      num = 2
      print(type(name))
      print(type(num))
    输出：
    <class 'str'>
    <class 'int'>

    数据转换 ---- 数据转换的方法有3个: str() ，int()，float()

    转换方向转换方法
    str --> intint() 文字类和小数点的字符串不能转换为int
    str --> floatfloat() 文字类的字符串不能转换为float
    int --> strstr()
    int --> floatfloat()
    float --> intint() 抹0取整
    float --> strstr()

  • Python 基础语法3

    if 条件判断

    1. 单项判断 ---- if 注意冒号下缩进
    2. 双向判断 ---- if...else...else需要一个平级的前提条件
    3. 多项判断 ---- if...elif...else...elif需要if，但不一定需要else

    if嵌套 ---- 条件套条件，额外条件要在基础条件下缩进

    特别强调if语句的缩进和冒号 
      JBTime = int(input('请输入小K的加班时间点：'))
      if JBTime > 10:
          print('小K 打车回家')
      elif JBTime >= 8:
          print('小K 坐地铁回家')
      else:
          print('小K 骑车回家')

    input()函数

      name = input('请输入你喜欢人的名字：')
      print(name)

    特点：

    1. 有问有答，有来有往。需要在终端输入信息。
    2. 函数好用，赋值第一。input()函数的结果必须赋值。
    3. 返回类型，必为str。不管输入的是整数1234，还是字符串，input()函数的输入值，都会被强制性的转换为字符串类型。
    4. 想要整数，源头转换。输入值需要整数时，input()函数的结果需要强制转换。

  • Python 基础语法5

    列表 (list)

      transformers = ['擎天柱', '大黄蜂', '救护车']

    以上 ['擎天柱', '大黄蜂', '救护车'] 就是一个列表。
    一个列表需要使用中括号[]将所有数据框起来，里面中的每一个数据叫做“元素”， 元素与元素之间使用逗号隔开。

    列表的打印

      list = ['李雷','韩梅梅',180,3.5]
      print(list)
    输出：['李雷', '韩梅梅', 180, 3]

    你会发现，列表的胸怀很宽广，各种类型的数据（字符串、整数、浮点数）都能放在里面。

    从列表中提取单个元素

    我们就可以通过列表名搭配下标方式来提取列表中的元素，读取元素中的内容。 
      transformers = ['擎天柱','大黄蜂','救护车']
      print(transformers[1])
    输出：大黄蜂

    注意：列表的下标是从0开始的

    从列表中提取多个元素

      transformers = ['擎天柱','大黄蜂','救护车','巨无霸福特','红蜘蛛']
      print(transformers[:])
      print(transformers[2:])
      print(transformers[:2])
      print(transformers[1:3])
      print(transformers[2:4])
    输出为：
    ['擎天柱', '大黄蜂', '救护车', '巨无霸福特', '红蜘蛛']
    ['救护车', '巨无霸福特', '红蜘蛛']
    ['擎天柱', '大黄蜂']
    ['大黄蜂', '救护车']
    ['救护车', '巨无霸福特']

    这种通过冒号来截取列表元素的操作叫切片，就是将列表的某个片段，通过切割下标的方式来提取多个元素。
    江湖秘籍：冒号左边空，就要从下标为0的元素开始取。右边空，就要取到列表的最后一个元素。后半句：冒号左边数字对应的元素要拿，右边的不动。

    修改列表元素

      transformers = ['擎天柱','大黄蜂','救护车']
      transformers[2] = '威震天'
      print(transformers)
    输出：['擎天柱','大黄蜂','威震天']

    给列表添加 / 删除元素

    需要用append()函数给列表增加元素

      transformers = ['擎天柱','大黄蜂','救护车','巨无霸福特','红蜘蛛']
      transformers.append('萨克巨人')
      print(transformers)
    输出：['擎天柱', '大黄蜂', '救护车', '巨无霸福特', '红蜘蛛', '萨克巨人']

    Python中删除列表元素需要用到del函数。

    del函数的语法是：del 列表名[元素的下标]。

      transformers = ['擎天柱','大黄蜂','救护车','巨无霸福特','红蜘蛛']
      del transformers[2]
      print(transformers)
    输出：['擎天柱', '大黄蜂', '巨无霸福特', '红蜘蛛']

    字典

    当需要存放键值对数据的时候，就需要用到字典。
    字典和列表很相似，都有名称、赋值号、逗号做间隔，逗号间隔的还叫元素。
    而不一样的是，字典每个元素由键与值组成，而列表不是，字典外层使用大括号{ }， 列表外层使用中括号[ ]。

      fc = { '擎天柱': 95, '大黄蜂':90, '救护车':86, '巨无霸福特':80, '红蜘蛛':80 }

    字典的元素由键与值组成，组成形式 键:值，冒号还是英文的冒号。'擎天柱': 95， 我们把'擎天柱'称为键，把95称为值。
    你还可以使用len()函数来得到字典或列表的长度。len()函数括号里放字典或列表的名字。

      transformers = ['擎天柱','大黄蜂','救护车','巨无霸福特','红蜘蛛']
      fc = { '擎天柱': 95 ,'大黄蜂':90 ,'救护车':86, '巨无霸福特':80,'红蜘蛛':80  }
      print(len(transformers))
      print(len(fc))
    输出：5   5

    这里注意的是：字典中的键是唯一的，值是可以重复的。也就是说，字典里只能有一个'擎天柱'的键，但是却可以有两个'80'的值。

    从字典中提取元素

    现在，我们将擎天柱的战斗值取出来。和列表相似，使用[ ]，但是字典没有下标，只能通过键名提取，也就是：字典名[键]

      fc = { '擎天柱': 95 ,'大黄蜂':90 ,'救护车':86, '巨无霸福特':80,'红蜘蛛':80  }
      print(fc['擎天柱'])
    输出：95

    修改字典元素值

      fc = { '擎天柱': 95 ,'大黄蜂':90 ,'救护车':86, '巨无霸福特':80,'红蜘蛛':80  }
      fc['巨无霸福特'] = 50 #修改元素巨无霸福特的值为50
      print(fc)
    输出：{'擎天柱': 95, '大黄蜂': 90, '救护车': 86, '巨无霸福特': 50, '红蜘蛛': 80}

    给字典增加 / 删除元素

      fc = { '擎天柱': 95 ,'大黄蜂':90 ,'救护车':86, '巨无霸福特':80,'红蜘蛛':80  }
      del fc['巨无霸福特']  #删除元素：巨无霸福特
      print(fc)
      fc['巨无霸福特'] = 50 #增加元素：巨无霸福特
      print(fc)
    输出：
    {'擎天柱': 95, '大黄蜂': 90, '救护车': 86, '红蜘蛛': 80}
    {'擎天柱': 95, '大黄蜂': 90, '救护车': 86, '红蜘蛛': 80, '巨无霸福特': 50}

    我们可以看到，删除字典里键值对是使用del语句： del 字典名[键]，而新增键值对是使用：字典名[键]=值。

    在字典中嵌套列表

      zidian = {'name':['小号','小红','小兰'], 'age':['男','女','女']}
      print(zidian['name'])
      print(zidian['age'][1])
    输出：
    ['小号', '小红', '小兰']
    女
