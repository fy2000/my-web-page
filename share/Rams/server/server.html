服务端程序讲解

一、程序简介

  • 图标和名称

    DtDrvApp.exe

  • 角色：中介

  • 作用：客户端和服务器之间的桥梁

    1. 为客户端的查询提供服务。
    2. 加强安全管理

  • 服务能力

    1. 支持四个维度的指标查询(文件维度、栅格维度、小区维度、小区栅格维度)。
    2. 支持 2G、3G、4G、5G 等多种网络类型的指标查询
    3. 支持 语音、数据、扫频 等多种业务类型的指标查询
    4. 支持测试文件下载。
    5. 支持 IPV6 查询访问
