String类型
1.用string类型定义的变量可以直接赋值字符串，string a = "12345",也可以直接比较
2.值类型包括：数值类型、布尔类型、字符类型
3.布尔类型：只有true和false两个值，关键字为bool，计算机中占4个字节（4B），即32位存储控件，但是不能用"0"和"1"代替。
4.字符类型:保存单个字符，关键字为char,每个字符对象与Unicode字符集中的字符对应，赋值时用单引号
5.引用类型：在C#预定义的类型中，属于引用类型的有对象(object)类型和字符串(string)类型
  6.object类型：所有其它类型的基类型，所以可以把任何类型的值赋值给它
  7.string类型：是一种特别的引用字符，通过复制实现对其的赋值。
  8.字符串类型的特性：
a.不可变性---当你给字符串变量重新赋值的时候，老值并没有销毁，而是重新开辟一块空间存储。
b.字符串可以看作是char类型的一个只读数组
  9.类型的转换有两种：隐式转换、显示转换
隐式转换：需要转换的类型位数小于等于目标类型位数，系统自动进行转换。
显示转换：需要转换的类型大于目标类型的位数，此时需要人为的操作才能转换正确。
  10.类型转换也可以用Convert方法，格式：Convert.方法()；进行转化。也可以格式：标识符.方法()；
也可以用int.parse();
  11.StringBuilder类型的使用
a.引入命名空间System.Text： using System.Text;
b.通过new关键字初始化： StringBuilder mystr = new StringBuilder(字符串);
c.使用时和string类型一样
