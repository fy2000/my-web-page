1.C#不是读作“C#”，而是读作“C Sharp”，C#是运行在.net CLR上的，用于创建应用程序的一种高级的编程语言。可以编写桌面应用程序、Windows Store应用程序、Web应用程序、WCF服务等。
2..net不是读作“点net”，而是读作“dotnet”，是微软开发的一个框架，最新版本4.5。.net是一个庞大的代码库，分为多个不同的模块。它定义了基本的类型，被称为通用类型系统（CTS）；包含.net公共语言运行库（简称CLR，负责管理用.net开发的所有应用程序的执行）
3.OOP-----面向对象编程技术(Object Oriented Programming，OOP，面向对象程序设计)
4.通用类型系统-----------Common type system(CTS)
5.公共语言运行库---------Common language Runtime(CLR)
  6..net的两种交互模式：
a.C/S模式(客户机/服务器模式):Client/Server,客户端需要安装专门的客户端软件
b.B/S模式(浏览器/服务器模式):Brower/Server,客户端只需要安装一个浏览器就可以了
  7..net framework编写应用程序的过程：
a.使用某种.net兼容语言编写代码
b.把代码编译为CIL，存储在程序集中
c.在执行代码时，使用JIT编译器将代码编译为本机代码
d.托管到CLR环境下运行本机代码
  8.通用中间语言-----------Conmmon intermedia language(CIL)
  9.控制台程序----------控制台程序是为了兼容DOS程序而设立的，这种程序的执行就好像在一个DOS窗口中执行一样，没有自己的界面。只是通过字符串来显示或者监控程序。控制台程序常常被应用在测试、监控等用途，用户往往只关心数据，不在乎界面。
  10.命名空间是.net中提供应用程序代码容器的方式，这样就可以唯一的标识代码及其内容。命名空间是以层的形式存在的，多层时用"."隔开。
  11.标识符----在编程语言中，标识符就是程序员自己规定的具有特定含义的词，比如类名称，属性名称，变量名等。只能由大写字母、小写字母、数字、下划线组成。
  12.C#中如果关键字要作为标识符使用，需要在前面加上@符号，如:@class
  13..net中命名规范有两种，一种camelCase，一种PascalCase
  14.快速注示：Ctrl+K 松开K 按下C；  取消注示：Ctrl+K 松开K 按下U；
  15.变量名必须以字母、下划线或@开头，不要以数字开头，在C#中是区分大小写的，当用到英文单词的缩写时应当全部用大写
  16.定义常量需要在前面加上 const;格式：const 数据类型 常量名称=常量值；
  17.@的作用：a.忽略转义字符 b.让字符串跨行 c.可以让关键字变为标识符使用
  18.C#中预定义的数据类型有两种，a.值类型(存储在栈中) b.引用类型(存储在堆中)
  19.表达式-----将变量、字面值和运算符组合起来就得到了表达式。
  20.运算符按操作数分类：一元运算符、二元运算符、三元运算符（?:）
  21.运算符按运算类型分类：数学运算符、赋值运算符、关系运算符、布尔运算符、位运算符、其它运算符
  22.关系运算符的结果为true和false两种，返回一个bool类型。关系运算符又叫比较运算符
  23.关系运算符作用的操作数如果是两个字符串，那么只能用 ==和！=
  24.布尔运算符：&&、||、！、^、&、|
  25.位运算符：～、&、|、<<、>>、^
  26.字符串连接运算符：+；
  27.is运算符：用于动态的检查对象的运行时类型是否与给定类型兼容。 a = b is string;
  28.自己查阅运算符的优先级
