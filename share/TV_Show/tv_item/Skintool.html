二、如何添加UI元素

   通过学习SkinTool,已经对OSD有了一定的了解，但是还存在很多疑惑。以下是作业报告：
动态功能Silence选项的制作：

1.在skintool软件工具中右边的GWin list窗口中选中MAINFRAME-> MENU_OPTION_PAGE -> MENU_OPTION_PAGE_LIST

2.点击左边的Button,然后用鼠标在黑色区域画出菜单并命名为MENU_OPTION_SILENCE，再在此菜单下再新建MENU_OPTION_SILENCE_TEXT和MENU_OPTION_SILENCE_OPTION，如下图所示：

3.然后用Excel修改从SkinTool导出的字符表.txt文件，添加两行字符，并导入skintool，分别设置好属性。

4.保存并编译即可。

5. 在Source Insight中找到HWND_MENU_OPTION_SILENCE并设置如下：
    // 202 = HWND_MENU_OPTION_SILENCE
      {
             EN_ZUI_DEFAULTWINPROC, NULL, WS_VISIBLE|WS_FOCUSABLE,
             _MENU_OPTION_SILENCE_Navigation,
             _MENU_OPTION_SILENCE_KEY,
       },

    // 203 = HWND_MENU_OPTION_SILENCE_TEXT
      {
            EN_ZUI_DEFAULTWINPROC, NULL, WS_VISIBLE|WS_FOCUSABLE,
            NULL,
            NULL,
       },

    // 204 = HWND_MENU_OPTION_SILENCE_OPTION
      {
             EN_ZUI_DYNAMICTEXT_WINPROC, NULL, WS_VISIBLE|WS_FOCUSABLE,
             NULL,
             NULL,
       },

6.然后进入_MENU_OPTION_SILENCE_KEY并设置如下：
       static KEYEVENT _ZUI_TBLSEG _MENU_OPTION_SILENCE_KEY[] =
      {
      { VK_SELECT, EN_EXE_ONOFF_OPTION  },
      };

7.然后如下：
      case HWND_MENU_OPTION_SILENCE_OPTION:          
           if(stGenSetting.u8Silence)//msAPI_AUD_IsAudioMutedByUser()
            {
                     u16TempID=en_str_On;
             }
             else
            {
                     u16TempID=en_str_Off;
            }
            break;

8.接着找到EN_EXE_ONOFF_OPTION设置按键动作：
     case EN_EXE_ONOFF_OPTION:                
     {     
             if(stGenSetting.u8Silence)
                  stGenSetting.u8Silence = 0;
             else
                  stGenSetting.u8Silence = 1;        
      }

      MApp_ZUI_API_InvalidateWindow(HWND_MENU_OPTION_SILENCE_OPTION);
      return TRUE;

9.然后在SecureCRT中编译就可以了。
